# Repository Overview — template-service

## Purpose

The template-service stores notification templates and renders them with dynamic data. It provides a synchronous API consumed by notification-service during the notification flow.

## Tech Stack

- Runtime: Node.js
- Framework: Express
- Database: PostgreSQL
- Templating: Handlebars

## Getting Started

```bash
npm install
npm run dev
```

## Key Directories

- `src/` — Application source code
- `docs/` — Documentation and specifications
- `.kiro/` — Kiro specs and steering files
